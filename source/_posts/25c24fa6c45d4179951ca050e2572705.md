<https://github.com/hangwin/mcp-chrome>

## 安装

1. 从[github](https://github.com/hangwin/mcp-chrome/releases)上下载最新的chrome扩展

2. 全局安装mcp-chrome-bridge

```bash
# npm
npm install -g mcp-chrome-bridge

# 或pnpm

# 方法1：全局启用脚本（推荐）
pnpm config set enable-pre-post-scripts true
pnpm install -g mcp-chrome-bridge

# 方法2：如果 postinstall 没有运行，手动注册
pnpm install -g mcp-chrome-bridge
mcp-chrome-bridge register
```

3. 加载 Chrome 扩展

- 打开 Chrome 并访问 chrome://extensions/
- 启用"开发者模式"
- 点击"加载已解压的扩展程序"，选择 your/dowloaded/extension/folder
- 点击插件图标打开插件，点击连接即可看到mcp的配置

4. json

```json
{
  "mcpServers": {
    "chrome-mcp-server": {
      "type": "streamableHttp",
      "url": "http://127.0.0.1:12306/mcp"
    }
  }
}
```
