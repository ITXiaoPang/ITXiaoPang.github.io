一个模型上下文协议服务器，可访问您的 iTerm 会话。

<https://github.com/ferrislucas/iterm-mcp>

## 安装

```json
{
  "mcpServers": {
    "iterm-mcp": {
      "command": "npx",
      "args": [
        "-y",
        "iterm-mcp"
      ]
    }
  }
}
```
