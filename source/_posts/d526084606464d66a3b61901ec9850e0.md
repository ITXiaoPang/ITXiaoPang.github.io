直接在 Claude Code 中使用 Gemini 强大的分析能力，以节省代币并分析大文件。

<https://github.com/jamubc/gemini-mcp-tool>

## 安装

```bash
claude mcp add gemini-cli -- npx -y gemini-mcp-tool
```

或

```json
{
  "mcpServers": {
    "gemini-cli": {
      "command": "npx",
      "args": ["-y", "gemini-mcp-tool"]
    }
  }
}
```
