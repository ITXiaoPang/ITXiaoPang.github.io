## 强制设置为英语

```bash
defaults write com.google.Chrome AppleLanguages '(en-US)'
```

## 强制设置为中文

```bash
defaults write com.google.Chrome AppleLanguages '(zh-CN)'
```

## 重置默认值

```bash
defaults delete com.google.Chrome AppleLanguages
```
