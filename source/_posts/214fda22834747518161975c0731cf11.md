一个使用 AntV 生成图表的模型上下文协议服务器。我们可以使用这个 mcp 服务器进行图表生成和数据分析。

<https://github.com/antvis/mcp-server-chart>

## 安装

```json
{
  "mcpServers": {
    "mcp-server-chart": {
      "command": "npx",
      "args": [
        "-y",
        "@antv/mcp-server-chart"
      ]
    }
  }
}
```
