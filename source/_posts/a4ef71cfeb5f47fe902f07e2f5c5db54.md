一个使用 Playwright 提供浏览器自动化功能的模型上下文协议服务器。该服务器使 LLMs 能够与网页交互、截屏、生成测试代码、抓取网页内容并在真实浏览器环境中执行 JavaScript。

<https://github.com/executeautomation/mcp-playwright>

```json
{
  "mcpServers": {
    "Playwright": {
      "command": "npx",
      "args": [
        "-y",
        "@executeautomation/playwright-mcp-server"
      ],
      "env": {}
    }
  }
}
```
