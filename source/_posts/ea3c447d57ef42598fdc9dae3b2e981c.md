一个模型上下文协议（MCP）实现，能够与 MySQL 数据库进行安全交互。该服务器组件促进 AI 应用（主机/客户端）与 MySQL 数据库之间的通信，通过受控接口使数据库探索和分析更加安全和结构化。

<https://github.com/designcomputer/mysql_mcp_server>

## 安装

```json
{
  "mcpServers": {
    "MySQL Server": {
      "command": "uvx",
      "args": [
        "--from",
        "mysql-mcp-server",
        "mysql_mcp_server"
      ],
      "env": {
        "MYSQL_HOST": "localhost",
        "MYSQL_PORT": "3306",
        "MYSQL_PASSWORD": "123654qwe",
        "MYSQL_USER": "root",
        "MYSQL_DATABASE": "cmdb"
      }
    }
  }
}
```
